# 🛡️ Aditya Kumar Singh — Cybersecurity Portfolio

> *"Securing the digital world, one vulnerability at a time."*

A personal cybersecurity portfolio built with **pure HTML, CSS & JavaScript** — no frameworks, no dependencies. Features a matrix rain effect, terminal-style UI, embedded certificates, and a downloadable CV.

🔗 **Live Site:** [your-portfolio.vercel.app](https://your-portfolio.vercel.app)

---

## 👨‍💻 About

Final year B.Tech CSE student at **Lovely Professional University**, aspiring **Penetration Tester** ranked in the **Top 5% on TryHackMe globally**. This portfolio showcases my projects, certifications, skills, and achievements in cybersecurity.

## ✨ Features

- 🖥️ **Terminal-style Hero** — interactive typing animation with hacker aesthetic
- 🌧️ **Matrix Rain** — cursor-interactive matrix background
- 📁 **Projects Showcase** — 4 real cybersecurity & dev projects with GitHub links
- 🎓 **Certificates Gallery** — real certificate images with click-to-zoom lightbox
- 📄 **CV Preview & Download** — embedded resume with one-click PDF download
- 🏆 **Achievements** — TryHackMe Top 5% badge, CTF competitions
- 📱 **Fully Responsive** — works on all screen sizes
- ⚡ **Single File** — everything embedded in one `index.html`, no server needed

## 🚀 Deployment

This portfolio is deployed on **Vercel** via GitHub integration.

Every push to `main` branch triggers an automatic redeploy.

```bash
git add .
git commit -m "Update portfolio"
git push
# Vercel auto-deploys in ~30 seconds ✅
```
